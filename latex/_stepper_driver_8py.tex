\doxysubsection{Detailed Description}
This is a class program that interfaces with both the TMC2208 and the TMC4210. 

This program acts as the bridge between the microcontroll and the stepper motor drivers. This allows the mcu to send and read instructions from the stepper motor drivers. It has the capabilities of enabling the stepper motor drivers and handles the IC for the TMC4210. It is capable of changing the motor operating modes, set velocities, and positions.
