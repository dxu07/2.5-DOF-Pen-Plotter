\doxysubsection{Detailed Description}
Driver to be used with the pyb board on the nerf gun to control firing. 

This driver interacts directly with the pyb board. It manages the nerf guns integrated firing mechanisms as well as the IR sensor we used to determine if a dart has been fired or not.
