\doxysubsection{Detailed Description}
This is the main program that connects all the different tasks together. 

This utilizes the cotask to create a cooperative set of tasks for simulaneous actions. This means it takes in the usesr\+\_\+input task, the process\+\_\+hpgl task, and also the positioning task.
