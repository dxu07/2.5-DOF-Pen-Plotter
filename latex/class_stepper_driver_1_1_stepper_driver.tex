\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_stepper_driver_1_1_stepper_driver}{Stepper\+Driver}} class that interfaces both the \mbox{\hyperlink{class_stepper_driver_1_1_t_m_c4210}{TMC4210}} and the \mbox{\hyperlink{class_stepper_driver_1_1_t_m_c2208}{TMC2208}} together. 

This class creates objects that work with both \mbox{\hyperlink{class_stepper_driver_1_1_t_m_c4210}{TMC4210}} and the \mbox{\hyperlink{class_stepper_driver_1_1_t_m_c2208}{TMC2208}}. This allows for separate objects that can control the motors independently.
