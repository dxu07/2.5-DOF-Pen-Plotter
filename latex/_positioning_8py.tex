\doxysubsection{Detailed Description}
This program handle the stepper motor driving and the firing of the nerf gun. 

This program interacts directly with \mbox{\hyperlink{_positioning_8py}{Positioning.\+py}} and \mbox{\hyperlink{_nerf_driver_8py}{Nerf\+Driver.\+py}} to move and shoot the nerf gun. It takes in data from Process\+HPGL or commands from User\+Input to determine where to move the gun and if it should shoot.
