This function is a task that controls the motors and the nerf gun.

This function handles the SPI controlling of the stepper motors. It interfaces with the Stepper Driver to read and write commands to it. This allows it to control each motor independently. Additionally, it handles the firing of the nerf gun.

Parameters
queuesis the queue of shared data for positioning
pausedis the boolean determining whether the paused command has been sent
stoppedis the boolean determining whether the stopped command has been sent
